# Overrides

## Options

`property`:

- `margin`
- `padding`

`direction`:

- `top`
- `bottom`
- `vertical` - `top` and `bottom`

## Removing space

```
// For all devices
tna-!--no-[property]-[direction]
```

`device` options (optional):

- `large`
- `medium`
- `small`
- `tiny`

```
// For a specific device size
tna-!--no-[property]-[direction]-[device]
```

## Adding space

`size` options:

- `xs`
- `s`
- `m`
- `l`
- `xl`
- `xxl`

```css
tna-!--[property]-[direction]-[size]
```
